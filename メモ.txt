リコー三愛グループ健康保険組合の健診データ作成について

1.データを抽出する
　「A96 三愛グループ健診データ提出用」パターンを使用
　保存形式はタブ区切りのテキスト

2.データ変換をする
　データ抽出したファイルを
　「NwToRicohSanai.exe」へドロップすると
　データ変換したファイルが作成されます。
　（例：リコー三愛グループ健康保険組合健診データ20230609.csv）

3.ログフォイルを確認
　データ変換時に「log.txt」が作成されます。
　変換エラーがないか確認します。
　確認後、ログファイルは削除してよい


4.暗号化ツール（アタッシュケース）にて暗号化する
　PW:RS13023

5.CDラベルに記載する内容
・医療機関名
・受診日
・提出日
・レコード件数


※PSA CA125 CA19-9 CEA AFP(定量) シフラの陰・陽区分について
数値より算出している。基準値に変更があった際は変換プログラムを修正する事

2023年6月16日時点での各検査の基準値
PSA      : 0 ～  4.00
CA125    : 0 ～ 35.0
CA19-9   : 0 ～ 37.0
CEA      : 0 ～  5.0
AFP(定量): 0 ～ 10.0
シフラ   : 0 ～  3.5
